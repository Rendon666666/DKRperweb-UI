# vue3 + element-plus +vite

## 介绍

---

这是一个计划从零开始的个人博客 前端页面使用vue3 + element-plus +vite 技术实现

持续更新中

## Project Setup

下载依赖
```sh
npm install
```

### Compile and Hot-Reload for Development

启动
```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

打包
```sh
npm run build
```
